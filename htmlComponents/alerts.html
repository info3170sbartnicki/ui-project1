Example

For styling, use one of the three context classes (for example, .alert-danger). To disable it inline, use the Alert JavaScript plugin.

Danger alert example

Warning alert example

Success alert example
HTML
           
            <div class="alert alert-danger" role="alert">
                Danger alert example
            </div>
            <div class="alert alert-warning" role="alert">
                Warning alert example
            </div>
            <div class="alert alert-succes" role="alert">
                Success alert example
            </div>
CSS
           
            .alert-danger,
            .alert-warning,
            .alert-succes {
              padding: 20px;
              width: 60%;
              border-radius: 10px;
            }
