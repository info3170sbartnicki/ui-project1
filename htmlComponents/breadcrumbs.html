Example of a breadcrumb navigation menu:

  • Landing Page
  • Page 2
  • Page 3
  • Page 4

To add actual pages to the menu, replace "#" in the list href with the link and rename.

HTML
       
        <div class="breadcrumbSection">
            <ul class="breadcrumb">
                <a href="#">Landing Page</a>
                <a href="#">Page 2</a>
                <a href="#">Page 3</a>
                <a href="#">Page 4</a>
            </ul>
        </div>
CSS
       
        ul.breadcrumb {
          padding: 10px 16px;
          list-style: none;
          background-color: var(--secondary);
          border-radius: 10px;
        }
        
        ul.breadcrumb li {
          display: inline;
          font-size: 18px;
        } 
        
        ul.breadcrumb li + li:before {
          padding: 8px;
          color: var(--light);
          content: '>';
        }
        
        ul.breadcrumb li a {
          color: var(--main);
          text-decoration: none;
        }
        
        ul.breadcrumb li a:hover {
          color: var(--light);
          text-decoration: underline;
          transition: transform 0.2s;
        }
